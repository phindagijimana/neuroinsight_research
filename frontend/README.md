# NeuroInsight Research - Frontend

React + TypeScript + Vite frontend for the NeuroInsight Research platform.

## Development

```bash
npm install
npm run dev     # Start dev server with HMR
npm run build   # Production build
npm run preview # Preview production build
```

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS (custom navy theme)
- Recharts (charts)
- Niivue (NIfTI visualization)
