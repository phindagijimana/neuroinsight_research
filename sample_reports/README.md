# Sample Reports

This folder contains example PDF reports generated by NeuroInsight for reference purposes.

## About These Reports

These are sample outputs that demonstrate what users can expect when processing MRI brain scans through NeuroInsight. Each report includes:

- **Patient Information** (anonymized/synthetic for samples)
- **Hippocampal Volume Measurements** (left and right)
- **Volumetric Analysis** with statistical normative comparisons
- **Asymmetry Index** (lateralization assessment)
- **Clinical Interpretation** guidelines
- **Visual Representations** (hippocampal segmentation overlays)
- **Processing Metadata** (scan date, processing date, sequence info)

## Report Types

NeuroInsight generates comprehensive PDF reports that include:

1. **Executive Summary**
   - Key findings at a glance
   - Asymmetry index with clinical thresholds
   - Volume measurements (mm³)

2. **Detailed Metrics Table**
   - Absolute volumes (left/right hippocampus)
   - Normalized volumes (% of intracranial volume)
   - Z-scores relative to normative data
   - Asymmetry calculations

3. **Visual Analysis**
   - Coronal, sagittal, and axial views
   - Hippocampal segmentation overlays
   - Color-coded volume representations

4. **Clinical Context**
   - Interpretation guidelines
   - Reference ranges
   - Quality metrics

All sample reports contain:

## Generating Your Own Reports

To generate reports from your MRI scans:

1. Upload a T1-weighted MRI scan (`.nii` or `.nii.gz` format)
2. Wait for processing to complete (3-7 hours depending on scan resolution)
3. Click "Download Report" button in the NeuroInsight interface
4. Open the PDF to view comprehensive analysis

## Report Quality

The quality and accuracy of reports depend on:
- Input scan quality (resolution, contrast)
- Proper T1-weighted sequence acquisition
- Minimal motion artifacts
- Adequate hippocampal visibility

For best results, use:
- T1-MPRAGE or equivalent sequences
- 1mm³ isotropic resolution (recommended)
- Whole-brain coverage
- Properly oriented scans

## Technical Details

Reports are generated using:
- **FreeSurfer** hippocampal segmentation
- **ReportLab** PDF generation
- **Matplotlib** visualizations
- **Statistical normative data** for z-score calculations

## Adding Sample Reports

To add sample reports:

1. Generate reports from open-source MRI data (e.g., from OpenNeuro, OASIS)
2. Ensure patient information is anonymized or synthetic
3. Name files descriptively (e.g., `sample_report_typical.pdf`, `sample_report_atrophy.pdf`)
4. Commit and push to GitHub

Example naming:
- `sample_report_normal_volumes.pdf` - Typical healthy hippocampal volumes
- `sample_report_left_atrophy.pdf` - Example showing left hippocampal atrophy
- `sample_report_right_atrophy.pdf` - Example showing right hippocampal atrophy
- `sample_report_bilateral_atrophy.pdf` - Example showing bilateral atrophy
